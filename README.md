# Survivevania (code: Assets -> Scripts)
2D platformer (University project)
 
> Scenes: Main Menu, 4 Levels, End credit

## Mechanics
- 2D Player Movement
- Jump
- Coin Pickup

- User Interface
  - Health 
  - Coin pickup points
  
- Obstacles
  - Enemies
  - Water
  - Spikes
